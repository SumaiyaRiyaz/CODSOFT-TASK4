# 🎮 Rock, Paper, Scissors Game (Python)

## 📌 Project Overview

This is a simple command-line **Rock, Paper, Scissors** game built using Python. The user plays against the computer, which randomly selects its move. The game continues until the user decides to quit.

---

## 🚀 Features

* User vs Computer gameplay
* Random computer choice using Python's `random` module
* Input validation for correct choices
* Continuous gameplay with replay option
* Clear win/lose/tie result display

---

## 🛠️ Technologies Used

* Python
* Built-in `random` module

---


## 🎯 How to Play

1. Enter your choice: `rock`, `paper`, or `scissors`
2. The computer will randomly choose its move
3. The winner will be displayed
4. Choose whether to play again

---

## 🧠 Game Logic

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices → Tie

---

## 📸 Example Output

```
Welcome to Rock, Paper, Scissors!
Enter your choice (rock/paper/scissors): rock
You chose: rock
Computer chose: scissors
Congratulations! You won!
```
